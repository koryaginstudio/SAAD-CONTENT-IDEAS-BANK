content idea bank

Банк идей для контент-плана

Записывай сюда идеи для постов, Reels, сторис, разборов и кейсов. После отправки откроется GitHub, где нужно нажать финальную кнопку создания идеи.

Важно: GitHub откроет страницу с уже заполненной идеей. Там нужно нажать “Submit new issue”.

Как это работает

Сайт живет на GitHub Pages, а идеи хранятся в GitHub Issues. Без Google, без сервера, без платных сервисов.

1. Заполни формуИдея собирается в аккуратную карточку.
2. Откроется GitHubТам уже будет готовый текст заявки.
3. Нажми SubmitИдея сохранится в общем списке Issues.
