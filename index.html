Закидывай сюда идеи для постов, Reels, Stories, Telegram и других материалов. После отправки откроется GitHub, там нужно нажать кнопку создания issue, и идея сохранится в общем списке.

Как это работает

Форма собирает текст идеи и открывает GitHub Issue. Чтобы идея сохранилась, на GitHub нужно нажать финальную кнопку создания issue.

1
Заполни поля на этой странице.
2
Нажми “Добавить идею”.
3
На GitHub нажми “Create issue” или “Submit new issue”.

Важно

Для добавления идеи нужен GitHub-аккаунт. Просматривать список идей можно через кнопку “Смотреть все идеи”.

База

Все идеи хранятся в разделе Issues репозитория SAAD-CONTENT-IDEAS-BANK.
